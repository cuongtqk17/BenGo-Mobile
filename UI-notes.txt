2. NativeWind Known Issues & Alternatives (Tailwind classes to avoid):
- Translate classes (e.g., translate-x, translate-y): Use native React Native inline `style={{ transform: [{ translateX: ... }] }}`.
- Animation classes (e.g., animate-spin, animate-pulse): Use `react-native-reanimated` instead for complex animations.
- Box Shadow classes (e.g., shadow-md, shadow-lg): Use native React Native shadows (`elevation` on Android, `shadowColor`/`shadowOffset`/`shadowOpacity`/`shadowRadius` on iOS) in `style` objects.
